# Today's Me - 오늘의 나
> 일정, 할일 ,일기 등을 간편하게 관리할 수 있는 앱입니다.\
> 플러터 Provider 상태관리 방식과 SQLite 데이터베이스를 사용하였습니다.

## 주요 기능
> 일정, 할일, 일기 추가 수정 삭제 기능 



## 설치

```sh
git clone https://github.com/bymine/today-me.git
```
